Analisis Data Menggunakan Naive Bayes (GaussianNB)

Pengertian

Naive Bayes adalah metode klasifikasi berbasis probabilitas dengan asumsi bahwa setiap fitur bersifat independen. Pada penelitian ini digunakan Gaussian Naive Bayes karena data bersifat numerik (kontinu).

Dataset

Dataset yang digunakan adalah data sederhana untuk prediksi kelulusan mahasiswa berdasarkan:

  • Nilai
  • Kehadiran
  • Tugas

Implementasi Python (Sklearn)

from sklearn.naive_bayes import GaussianNB
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score
import numpy as np

# Dataset
X = np.array([
    [80, 90, 85],
    [60, 70, 60],
    [75, 80, 78],
    [50, 60, 55],
    [90, 95, 92],
    [45, 50, 50],
    [70, 85, 80],
    [65, 75, 70]
])

y = np.array(['Lulus', 'Tidak', 'Lulus', 'Tidak', 'Lulus', 'Tidak', 'Lulus', 'Tidak'])

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Model Gaussian NB
model = GaussianNB()
model.fit(X_train, y_train)

# Prediksi
y_pred = model.predict(X_test)

# Evaluasi
akurasi = accuracy_score(y_test, y_pred)

print("Prediksi:", y_pred)
print("Akurasi:", akurasi)

Hasil Analisis

Model berhasil melakukan klasifikasi dengan tingkat akurasi yang cukup baik. Gaussian Naive Bayes cocok digunakan untuk data numerik karena menggunakan distribusi normal (Gaussian).

Kesimpulan

Berdasarkan hasil analisis, metode Gaussian Naive Bayes dapat digunakan untuk klasifikasi data numerik dengan baik. Model ini sederhana, cepat, dan memiliki performa yang cukup tinggi meskipun dengan dataset kecil.
