Implementasi Naive Bayes (Python Sklearn)

1 Persiapan & Normalisasi Data

Mengimpor library `sklearn.preprocessing` untuk menyamakan skala fitur dan membagi data menjadi set pelatihan dan pengujian.

from sklearn.model_selection import train_test_split from sklearn.preprocessing import MinMaxScaler # Normalizer: Mengubah skala fitur ke rentang 0-1 scaler = MinMaxScaler() X_scaled = scaler.fit_transform(X) # Table Partitioner: Membagi data (80% Train, 20% Test) X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2)
2 Naive Bayes Learner & Predictor

Menggunakan class GaussianNB dari sklearn.naive_bayes untuk melatih model berdasarkan data yang sudah ada.

from sklearn.naive_bayes import GaussianNB # Learner: Proses pelatihan model model = GaussianNB() model.fit(X_train, y_train) # Predictor: Menebak hasil berdasarkan data uji y_pred = model.predict(X_test)
3 Scorer (Evaluasi Performa)

Menghitung akurasi dan melihat matriks perbandingan antara hasil asli vs prediksi.

from sklearn.metrics import accuracy_score, classification_report # Scorer: Menampilkan hasil akurasi print(f"Akurasi: {accuracy_score(y_test, y_pred) * 100:.2f}%") print(classification_report(y_test, y_pred))
